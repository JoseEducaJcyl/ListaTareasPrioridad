# Gestor de Tareas (To-Do List)

## Descripción
Aplicación de consola en Java que permite gestionar una lista de tareas mediante un menú interactivo. Ideal para organizar actividades pendientes de forma sencilla.

## Funcionalidades
- **Añadir tarea al final**: Agrega una nueva tarea al final de la lista
- **Insertar tarea en posición**: Coloca una tarea en una posición específica
- **Eliminar tarea por posición**: Borra la tarea que se encuentra en el índice indicado
- **Mostrar tareas**: Visualiza todas las tareas con sus números de posición
- **Validación de posiciones**: Controla que las posiciones ingresadas sean válidas

## Tecnologías
- Java
- ArrayList
- Scanner
- Estructuras de control (while, switch)
